# 宠物服务平台 MVP 原型 (Pet Service Platform MVP)

本项目用于构建宠物服务产品的 MVP 原型。
**目标**：实现页面样式及简单的交互跳转，无需实现完整的后端业务逻辑。

## 产品说明

### 功能拆分与优先级

| 功能拆分 | 子功能 | 优先级 | 说明 |
| :--- | :--- | :--- | :--- |
| **C 端（宠物主）** | **宠物档案创建** | **P0** | 必填项精简：品种、年龄、性别、照片、特殊备注 |
| | **服务类型选择** | **P0** | 喂养 / 遛狗 |
| | **时间/日期选择** | **P0** | |
| | **地址定位** | **P0** | |
| | **订单支付** | **P0** | 微信支付（模拟交互） |
| | *说明* | | 仅支持单宠物、单次、单一服务，排除复杂组合 |
| **B 端（宠托师）** | **实名认证** | **P0** | 身份证 + 人脸 |
| | **手动接单** | **P0** | 无智能派单，列表选择 |
| | **服务状态更新** | **P0** | 接单 → 到达 → 完成 |
| | **服务后拍照上传** | **P0** | 水印相机 |
| | *说明* | | 暂不支持录像、直播、路线优化；靠人工协调 |
| **基础安全保障** | **双方手机号认证** | **P0** | 登录注册，逻辑复用 |
| | **宠物主人设置紧急联系人** | **P0** | 可选 |
| | **服务完成强制上传 2 张照片** | **P0** | 宠物状态 + 门窗关闭 |
| **基础订单管理** | **订单状态查看** | **P1** | 待接单 / 服务中 / 已完成 |
| | **服务评价** | **P1** | 1–5 星 + 文字 |
| **简易后台管理** | **人工客服介入机制** | **P1** | 30分钟未接单，客服电话联系 |
| | **服务人员审核** | **P1** | 人工后台审核 |

---

## 技术栈

- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS
- **Language**: TypeScript

## Getting Started

First, run the development server:

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
